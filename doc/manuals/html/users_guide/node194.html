Basics

This section summarizes basic features that are common to both client and implementation bindings. Conventions used to protect the global name space are described in Subsection 9.2.1, while those associated with the generation of subroutines from methods are given in Subsection 9.2.2. Translations between SIDL and native Fortran 90 constructs are described in Subsection 9.2.3. Finally, the process of casting between different types is illustrated in Subsection 9.2.4.
