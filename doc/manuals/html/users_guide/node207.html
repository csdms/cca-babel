Overloaded methods

Examples of calls to SIDL overloaded methods are based on the overload_sample.sidl file shown in Section 5.7. Recall that the file describes three versions of the getValue method. The first takes no arguments, the second takes an integer argument, and the third takes a boolean. Each is called in the following code snippet 


  use sidl
  use Overload_Sample
  type(Overload_Sample_t)         :: t
  type(sidl_BaseInterface_t)      :: exception
  logical                         :: b1, bretval
  integer (kind=sidl_int)  :: i1, iretval

  call new(t, exception)

  call getValue (t, iretval, exception)
  call getValueInt (t, i1, iretval, exception)
  call getValueBool (t, b1, bretval, exception)
