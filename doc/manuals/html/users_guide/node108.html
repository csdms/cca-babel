Contract Enforcement

Checks to enforce contracts are automatically generated by the Babel compiler along with the language interoperability middleware. This approach allows interface contracts to be defined once in the specification and applied to all associated implementations in any of the supported languages.

WARNING:The routines used to specify contract enforcement options are not currently available in all language bindings. Refer to language-specific chapters for more information.
