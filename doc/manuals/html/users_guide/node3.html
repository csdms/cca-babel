Scope of this Manual

This document is intended as an introduction and tutorial on the use of Babel tools for the generation and use of component software. The Babel tools were designed specifically for scientific applications, therefore most of the examples and exercises here also deal with scientific applications.

This manual assumes the reader is a programmer who is proficient in two or more of the following languages: C, C++, FORTRAN 77, Fortran 90, Java, or Python. Furthermore, this manual assumes the reader is familiar with the SPMD1.2 programming model that pervades the scientific computing community. Knowledge of and experience with MPI programming is helpful, but not strictly required.
