Introduction

This chapter provides an overview of the Java bindings for SIDL. Common aspects of the bindings, such as the mapping of SIDL data types to their Java counterparts, are presented in Section 10.2. Issues of concern to callers written in Java are addressed in the client-side discussion in Section 10.3, while issues for callees appear in the implementation-side discussion in Section 10.4.
