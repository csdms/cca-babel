Basics

This section summarizes basic features that are common to both client and implementation bindings. Subsection 7.2.1 describes conventions used to establish name spaces, while those associated with the generation of subroutines from methods are given in Subsection 7.2.2. The mapping of fundamental and key SIDL types is given in Subsection 7.2.3. Finally, casting between different types is discussed in Subsection 7.2.4.
